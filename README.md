# Aplikasi Data Mahasiswa (Console Python)

## Deskripsi
Aplikasi ini dibuat menggunakan Python berbasis console untuk:
1. Menambah data mahasiswa
2. Menampilkan semua data
3. Menghitung rata-rata nilai
4. Menyimpan dan membaca file CSV
5. Mencari data berdasarkan nama

6. ## Cara Menjalankan
7. pilih menu
8. jika 1 maka tambahkan data mahasiswa nya
9. jika 2 maka akanditampilkan semua datanya
10. jika3 maka menghitung rata rata nilai
11. jika 4 maka untuk mencari data
12. dan jika 5 akan keluar programnya

13. ## Format Data CSV
nama, tugas, uts, uas
